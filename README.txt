
 # Pokedex
 **********************************************************************************************

 it's a website that groups pokemones according to their eggs-group,it contains 4 pages:

 * welcome  		: simple page with buttons multilanguage and button start.
 * index 		    : list of eggs group.
 * listPokemon		: list of pokemon in a selected group.
 * detailPokemon	: detail of pokemon selected.

 ## basic technologies
 **********************************************************************************************

 * [JDK]			            (https://www.oracle.com/java/technologies/downloads/): Version 11 
 * [sparkjava]			        (https://sparkjava.com/): Version 2.9.3
 * [spark-template-handlebars]	(https://mvnrepository.com/artifact/com.sparkjava/spark-template-handlebars): Version 2.7.1
 * [GSON]			            (https://github.com/google/gson): Version 2.8.6
 * maven and eclipse are required for best usage	

 ## Installation
 **********************************************************************************************

 $ git clone https://github.com/abdess-amad/pokedex.git

 ## API
 **********************************************************************************************
 * this website consumes pokeApi, it is aRESTful API interface to highly detailed objects
 * built from thousands of lines of data related to Pokémon.Using this website, you can consume information on Pokémon, 
 * their moves, abilities, types, egg groups and much, much more.for more information https://pokeapi.co/.
